# logical-expressions
Logical Expressions

##Executar a classe principal App.java e o resultado sairá no console

* Duplo Implica      =
* Implica            >
* OU                 |
* E                  &
* Negacao            !
* Falso              F
* Verdadeiro         V


##Exemplos:
* (p>q)&(q>r)>(p>r)F     -   Tautologia
* p|!pF                  -   Tautologia
* (p&q)>(p=q)F           -   Tautologia
* p&!pF                  -   Não é tautologia
* (p|!q)>(p&q)F          -   Não é tautologia
